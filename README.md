## Installation

1. Install [Docker](https://www.docker.com/)
2. Clone this repository
    ```bash
    git clone https://github.com/nell-shark/spring-boot-blog
    ```
3. Run all containers
   ```bash
    docker-compose up
    ```
4. Go to [localhost:8080](http://localhost:8080)
